Словесный алгоритм программы №1:
1.	Начало программы.
2.	Открытие файла "students.txt" для чтения.
3.	Если файл не открылся, вывод сообщения об ошибке и завершение работы.
4.	Создание пустого однонаправленного списка (головной указатель headptr = NULL).
5.	Цикл чтения данных из файла, пока не достигнут конец файла:
   •	Чтение строки с ФИО студента.
   •	Если строка пустая или достигнут конец файла, выход из цикла.
   •	Чтение числа (средний балл) и игнорирование символа новой строки.
   •	Чтение строки с адресом.
   •	Создание нового узла списка, копирование данных в него.
   •	Добавление узла в конец списка.
6.	Закрытие файла.
7.	Вывод исходного списка на экран (функция pechpisok).
8.	Сортировка списка методом пузырька (функция sort):
   •	Повторять, пока выполняются обмены:
     o	Пройти по списку, сравнивая адреса соседних узлов.
     o	Если адрес текущего узла больше адреса следующего, обменять данные между узлами.
   •	Завершить, когда за весь проход не было ни одного обмена.
9.	Вывод отсортированного списка на экран.
10.	Конец программы.
Словесный алгоритм программы №2:
1.	Начало программы.
2.	Создание пустого вектора вещественных чисел.
3.	Запрос у пользователя количества элементов n.
4.	Если n <= 0, вывод сообщения об ошибке и завершение программы.
5.	Цикл от 0 до n-1:
    o	Запрос у пользователя вещественного числа.
    o	Добавление числа в конец вектора.
6.	Вывод исходного вектора на экран.
7.	Инициализация переменной maxIndex = 0 (индекс максимального элемента).
8.	Цикл от 1 до последнего индекса вектора:
    o	Если текущий элемент больше элемента с индексом maxIndex, обновить maxIndex.
9.	Обмен местами элемента с индексом 0 и элемента с индексом maxIndex с помощью swap.
10.	Вывод измененного вектора на экран.
11.	Вывод сообщения о том, какой элемент перемещен на первую позицию.
12.	Конец программы.
